== Description ==

[Groups File Access](https://www.itthinx.com/shop/groups-file-access) allows to restrict user access to files by group membership.

== Installation ==

1. Go to *Plugins > Add New > Upload* and choose the plugin's zip file, or extract the contents and copy the `groups-file-access` folder to your site's `/wp-content/plugins/` directory.
2. Enable the plugin from the *Plugins* menu in WordPress.
